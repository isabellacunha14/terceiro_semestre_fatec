# Terceiro Semestre - FATEC

Repositório **terceiro_semestre_fatec** contendo exercícios, projetos e trabalhos desenvolvidos durante o **terceiro semestre** do curso de graduação na **FATEC**.

O objetivo deste repositório é registrar a evolução do aprendizado ao longo do semestre, reunindo atividades práticas, códigos e projetos das disciplinas estudadas.

---

## 📚 Estrutura do Repositório

Os conteúdos estão organizados por disciplina, cada uma em sua própria pasta:

```
estrutura_de_dados/
programacao_orientada_objetos/
banco_de_dados_relacional/
```

Cada pasta pode conter:

* Exercícios de aula
* Trabalhos acadêmicos
* Projetos práticos
* Anotações ou materiais de apoio

---

## 💻 Tecnologias Utilizadas

Durante este semestre estão sendo utilizadas principalmente as seguintes tecnologias:

* **C** – Estruturas de dados e algoritmos
* **Java** – Programação Orientada a Objetos
* **SQL** – Modelagem e manipulação de bancos de dados relacionais

---

## 🎯 Objetivo do Repositório

* Registrar a evolução acadêmica durante o semestre
* Organizar os códigos desenvolvidos nas disciplinas
* Servir como material de consulta futura
* Demonstrar conhecimento técnico em desenvolvimento e banco de dados
